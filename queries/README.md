## Results Summary

- Identified a strong concentration of customer value among a small loyal segment
- Found that severe delivery delays reduce review scores by ~57%
- Discovered that only ~4.7% of marketing leads convert into active, revenue-generating sellers

## How to Use This Repository
- SQL queries are available in the `queries/` folder
- Each query is documented with the business problem it solves
- Dataset can be accessed from Kaggle (Olist Brazilian E-Commerce Dataset)
